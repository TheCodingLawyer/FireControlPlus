# GitHub Release Instructions for FireControl v1.0.0

## Overview

The repository is ready for release on GitHub. Follow these steps to publish the v1.0.0 release.

## Steps to Create GitHub Release

1. Go to: https://github.com/TheCodingLawyer/FireControlPlus/releases

2. Click "Create a new release"

3. Select tag: Choose v1.0.0 from dropdown

4. Fill in Release Title:
   FireControl v1.0.0

5. Fill in Description:
   Copy the content from RELEASE_NOTES.md

6. Upload Files:
   Attach: build/libs/FireControl-1.0.0.jar
   
   Path: C:\Users\ghost\Desktop\Work on Plugins\BanManager\BanManager\FireControl\build\libs\FireControl-1.0.0.jar

7. Check "This is a pre-release" - No, this is a stable release

8. Click "Publish release"
